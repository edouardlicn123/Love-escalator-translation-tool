二、获取 Groq API Key

步骤 1: 注册 Groq 账号

  1. 访问 console.groq.com
  2. 点击 "Sign Up" 注册账号
  3. 可以使用 Google 账号快速登录

步骤 2: 创建 API Key

  1. 登录后进入 API Keys 页面
  2. 点击 "Create API Key" 按钮
  3. 输入一个名称（如 "translation-tool"）
  4. 点击 "Submit" 创建
  5. 立即复制 API Key（只显示一次！）

步骤 3: 在工具中配置

  1. 打开翻译质量检查工具
  2. 在 Groq API 配置 区域
  3. 粘贴 API Key 到输入框
  4. 选择模型（推荐 llama-3.1-8b-instant 速度更快）
  5. 点击 "保存配置"

三、测试 API

使用 curl 测试

在终端运行以下命令（将 YOUR_API_KEY 替换为你的密钥）：

curl -X POST "https://api.groq.com/openai/v1/chat/completions" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "llama-3.1-8b-instant",
    "messages": [{"role": "user", "content": "こんにちは、お元気ですか？"}]
  }'

成功的响应：

{
  "choices": [{
    "message": {
      "content": "你好，你好吗？"
    }
  }]
}

使用测试脚本

# 运行测试脚本
chmod +x test_groq.sh
./test_groq.sh YOUR_API_KEY

# 或指定模型
./test_groq.sh YOUR_API_KEY llama-3.3-70b-versatile

四、使用检查功能

1. 建立数据库

点击页面上的 "建立数据库" 按钮，同步所有翻译条目到本地数据库。

2. 选择检查规则

勾选需要检查的问题类型：

  • cn为空 - cn字段为空或仅空格
  • 未翻译占位符 - 包含「」或……等占位符
  • 长度异常 - 中文字数与原文字数比例异常
  • 相同内容 - cn与jp完全相同
  • 数字占位符 - 包含 [NUM] 等未替换占位符
  • 翻译质量低 - 包含机械翻译痕迹

3. 查看和修正翻译

  1. 点击 "确认并重新检测" 按钮
  2. 使用 "上一条" / "下一条" 导航
  3. 查看 AI 提供的翻译建议
  4. 在文本框中编辑翻译
  5. 点击 "保存→跳转" 保存并跳到下一条

4. 备份数据

定期点击 "备份数据库" 按钮，防止数据丢失。

五、常见问题

Q: 翻译失败怎么办？

  • 检查 API Key 是否正确（不要有多余的空格）
  • 确认 API Key 还有额度（Groq 每日 14,400 次）
  • 查看浏览器控制台或后端日志的错误信息

Q: 速度变慢怎么办？

  • 尝试切换到 llama-3.1-8b-instant 模型
  • 等待几分钟后重试（可能达到速率限制）
  • 检查网络连接

Q: 如何选择模型？

  • llama-3.1-8b-instant - 速度快，免费额度内推荐
  • llama-3.3-70b-versatile - 质量更高，适合复杂翻译

Q: API Key 过期吗？

Groq API Key 不会过期，但如果达到每日额度限制，需要等到次日或使用付费版本。
