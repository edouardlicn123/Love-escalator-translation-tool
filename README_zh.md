## 关于 Love Escalator

**Love Escalator**（ラブ・エスカレーター）是一款于 **1998年4月** 发售的日本成人视觉小说游戏，对应 **NEC PC-9800 系列**（PC-98）平台。由 **Umitsuki Production**（海月制作所）开发，因其发售时间正值 PC-98 时代末期常被称为"PC-98 最后的エロゲ"。

### 游戏剧情

主角黑崎隆向好友胁谷彻的请求，帮助他追求曾在上中学时暗恋的河合理恵。随着与理恵的接触，隆向发现理恵中学时也对自己有好感。两个人互相隐瞒着对对方的心意，暗中开始交往。游戏讲述了在友情与爱情之间摇摆的复杂情感故事。

### 游戏特点

- **类型**: 恋爱冒险 ADV / 视觉小说
- **发售日期**: 1998年4月17日
- **平台**: PC-98 (DOS) / Windows 98
- **开发商**: Umitsuki Production
- **特色**: 丰富的动画演出，被评价为 PC-98 末期的代表作之一

## 功能特点

### 核心功能

| 功能 | 说明 |
|------|------|
| 🔍 **搜索过滤** | 搜索翻译内容，按状态筛选 |
| ⏭️ **导航跳转** | 跳转到指定ID，上一条/下一条 |
| 🤖 **AI翻译** | 集成 Groq API 提供智能翻译建议 |
| 💾 **数据同步** | SQLite 数据库与 JSON 源文件双向同步 |
| 📋 **批量操作** | 备份数据库、导出数据 |

### 数据管理

- **翻译状态跟踪**: 标记翻译为已确认/待处理
- **数据统计**: 查看总数、已翻译、已确认数量
- **修改历史**: 保留所有修改记录

### 界面功能

- **响应式设计**: 简洁现代的界面
- **字符计数**: 实时显示翻译字数
- **XSS 防护**: 安全渲染内容
- **请求超时**: 防止请求卡死

## 环境要求

### 系统要求

- **操作系统**: Linux / Windows (推荐使用 WSL)
- **Python**: 3.8+
- **网络**: 需要网络访问以使用 AI 翻译功能（Groq API）

### Python 依赖

- `requests` - HTTP 请求库
- `sqlite3` - 内置数据库支持
- 标准库模块: `json`, `http.server` 等

## 安装部署

```bash
# 克隆仓库
git clone https://github.com/your-repo/Love-escalator-translation-tool.git
cd Love-escalator-translation-tool

# 安装依赖（如需要）
pip install requests

# 启动服务器
python3 check_translation.py --port 5382
```

或使用提供的 shell 脚本：

```bash
chmod +x start.sh
./start.sh
```

## 使用方法

### 网页界面

在浏览器中打开：

```
http://localhost:5382
```

### 主要操作

1. **初始化数据库**: 点击"同步原文"同步原始日语文本
2. **搜索**: 使用搜索框查找特定翻译
3. **过滤**: 勾选"显示已确认"切换显示状态
4. **编辑**: 在编辑区域修改翻译
5. **保存**: 点击"保存→跳转"保存并跳转到下一条
6. **导航**: 使用按钮或键盘快捷键导航

### API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 主页面 |
| `/api/next` | GET | 获取下一条未确认记录 |
| `/api/update` | POST | 更新翻译 |
| `/api/translate` | POST | AI 翻译 |
| `/api/db/sync-jp` | POST | 从源文件同步原文 |
| `/api/db/sync-cn` | POST | 从源文件同步翻译 |
| `/api/search` | GET | 搜索翻译 |

## 兼容性说明

### PC-98 游戏支持

本工具专为 **PC-98 时代游戏**（1990年代的日本视觉小说）设计，可能兼容以下类型：

- 使用相似 JSON 结构翻译数据的游戏
- 需要翻译管理的日本视觉小说
- 已提取为文本格式的 PC-98 时代游戏

### 数据结构要求

工具期望 `table.json` 如下格式：

```json
{
  "filename.eqa": [
    {
      "jp": "日语文本",
      "cn": "中文翻译",
      "jpHex": "...",
      "cnHex": "...",
      "startIdx": 123,
      "endIdx": 456
    }
  ]
}
```

### 扩展到其他游戏

由于本工具的数据结构设计具有通用性，理论上可以用于：
- 其他 PC-98 视觉小说的翻译整理
- 使用类似 JSON 格式的翻译数据管理
- 任何需要批量翻译质量检查的项目
