# 列表功能设计文档

## 需求概述

在跳转/搜索/过滤列和翻译区之间新增一列列表，用于快速导航。

## 布局

```
┌──────────┬──────────┬─────────────────────────┐
│  跳转     │ ┌──────┐│                         │
│ [输入框]  │ │列表  ││     翻译区              │
│ [按钮]    │ │(可拖)││                         │
├──────────┤ │      ││                         │
│  搜索     │ │ID|原 ││                         │
│ [输入框]  │ │───  ││                         │
│ [按钮]    │ │000001││                         │
├──────────┤ │000002││                         │
│  过滤     │ │000003││                         │
│ ☑显示已确认│ │000004││                         │
│ [确认]    │ │...  ││                         │
│          │ └────┘│                         │
└──────────┴──────────┴─────────────────────────┘
```

## 列表规格

- **列宽**：约 120px
  - ID列：6位数字宽度（约 50px）
  - 原文列：4个中文字宽度（约 70px）
- **高度**：可滚动
- **滚动条**：浏览器原生滚动条

## 过滤逻辑

| 状态 | 复选框 | 显示内容 | 跳转位置 |
|------|--------|----------|----------|
| 默认 | 不打勾 | is_fixed=0 (未确认) | id最小 |
| 确认点击 | 打勾 | is_fixed=0 + is_fixed=1 | id=1 |
| 确认点击 | 不打勾 | is_fixed=0 (未确认) | id最小 |

## API 设计

### GET /api/next

**参数**：
- `show_fixed`: 1 表示显示全部，其他表示显示未确认

**示例**：
```
# 显示未确认
/api/next

# 显示全部
/api/next?show_fixed=1
```
